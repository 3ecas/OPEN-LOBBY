Visão Geral

Creative Cards É um jogo de cartas sobre produção audiovisual e media. Os jogadores angariam recursos para concluir projetos (cartazes, posts, reels, podcast, documentário, festival).
O primeiro a chegar a 20 pontos em projetos concluídos ganha.

Objetivo

Completar projetos usando cartas de Recurso. Cada projeto vale pontos conforme o tamanho:

  • Pequenos: 2|3 recursos " 2|4 pontos
  • Médios: 4|5 recursos " 5|7 pontos
  • Grandes: 6|10 recursos " 8|10 pontos

Setup

  1. Baralha o Baralho Principal (Recursos + Ações).
  2. Baralha o Baralho de Projetos em separado.
  3. Cada jogador recebe 5 cartas do baralho principal.
  4. Cada jogador recebe 3 projetos, escolhe 1 e devolve 2 para o fundo.
  5. Define quem começa. Joga-se por turnos.

Turno

  1. Podes jogar quantas cartas quiseres da mão.
  2. Só podes comprar 1 carta por turno.
  3. Podes alocar recursos ao teu projeto (saem da mão e ficam no projeto).
  4. Se tiveres todos os recursos (ou estiveres a usar Gambiarra), podes concluir o projeto.
  5. Depois de concluir, escolhes novo projeto (3 opções aleatórias).

Trocar 3 cartas iguais

Uma vez por turno:

  • Descarta 3 cartas de recurso iguais da tua mão;
  • Escolhe 1 recurso qualquer para receber em troca.

Cancelamento de Projeto

Ao tirar a carta Cliente cancelou o projeto:

  • O teu projeto atual é cancelado imediatamente;
  • Perdes todos os recursos alocados nesse projeto;
  • Não ganhas pontos;
  • Escolhes um novo projeto para começar.
